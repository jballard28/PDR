## Loading Data
You will need summary statistics that have the following columns in this exact order:
1. SNP: SNP rsID
2. A1: effect allele
3. A2: other allele
4. N: sample size
5. CHISQ: squared Z-score
6. Z: Z-score (note that it's sign is with respect to A1)
